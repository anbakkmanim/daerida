# Daerida

## Run Server In Local

1. 프로젝트 루트 폴더로 간다.

2. cmd에서 다음 명령어를 실행한다.

   ```bat
   > php -S localhost:포트번호    // 대소문자를 구분한다. 주의!
   ```

   포트가 사용 중일 수 있으니 오류가 나면 다른 포트로 시도해보자.
